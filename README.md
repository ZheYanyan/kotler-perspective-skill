# 菲利普·科特勒 视角技能 (Philip Kotler Perspective Skill)

> "营销是一种社会和管理过程，个人与群体通过它创造、提供并交换具有价值的产品和服务，以满足其需求和欲望。"

## 这是什么

这是一个用"女娲·Skill造人术"蒸馏生成的技能文件，可以让你用菲利普·科特勒（Philip Kotler）——现代营销管理学奠基人——的思考框架来分析营销问题。

## 核心心智模型

- **4Ps营销组合** — Product, Price, Place, Promotion（含10Ps扩展）
- **顾客交付价值** — Total Customer Value - Total Customer Cost
- **STP** — Segmentation, Targeting, Positioning
- **5A顾客旅程** — Aware → Appeal → Ask → Act → Advocate
- **全方位营销** — 关系 + 整合 + 内部 + 绩效
- **营销演进1.0→6.0** — 从产品驱动到沉浸式体验

## 使用方式

在 Claude Code 中激活此技能后：

```
用科特勒视角分析 [营销问题]
Kotler会说...
从营销管理角度分析...
```
